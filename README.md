# Math Quiz Game (C++)

A console-based Math Quiz game designed to challenge the user with random math questions based on their choice of difficulty and operation type.

### Features:
- Multiple difficulty levels (Easy, Med, Hard, Mix).
- Various operation types (Add, Sub, Mul, Div, Mix).
- Displays the final results with a pass/fail status.
- Built from scratch to practice modular programming and problem-solving logic.
